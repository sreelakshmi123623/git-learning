>git --version [command for checking git version]
>git config --list [git configuration list]
>git config --global user.name "username" [command for adding user name]
>git config --global user.email "useremail@gmail.com" [command for adding user email]
>git init [initializing git in present working directory]
>git add -A [command for adding all untracked files to staging area]
>git commit -m "commit msg" [adding stages files to localrepository]
>git status[list all untracked and modified files]
>git log [command for listing commit history]
>git remote add name remote_path
>git push origin master [for push from localrepositoryto remoterepository (master is the branch name)]
>git remove -v[list all remotes(can set any name here it is eg;origin)]
>git branch [list all branches (master)]
